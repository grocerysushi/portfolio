# Professional Portfolio

React + Vite portfolio template with sections for hero, GitHub repositories, featured projects, and resume content.

## Quick Start

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set your GitHub username:
   `VITE_GITHUB_USERNAME=your-github-username`
3. Run locally:
   `npm run dev`

## Build

- Type-check: `npm run lint`
- Production build: `npm run build`

## Customize

Update profile content in `src/App.tsx`:
- `PROFILE` for name, title, email, resume URL, and GitHub username fallback
- `PROJECTS` for featured work
- `EXPERIENCE` and `SKILLS` for resume details
